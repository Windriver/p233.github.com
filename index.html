Peter's Playground

嗨，我是 Peter，这是我的前端技术博客，分享资讯，积累经验，深度思考。。。


最近对页面做了全新的改版，目标是提供更简洁、舒适的阅读体验，同时也是一次与访客交互的思考。主要功能已基本完成，有三处问题暂时未能解决，能力不足一时半刻怕是解决不了，请大家在访问的时候注意：

  1. 写给 post 内容的 javascript 不能在页面载入时、或刷新后执行
  2. Disqus 不能根据静态地址获取评论内容，即同一篇 post 的 ajax 地址不同载入的评论内容也不同
  3. 令浏览器支持 ajax 历史记录
  4. 全屏模式在 Safari/Opera 下不工作，这只是插件原因，有时间会试着重写这一部分

各位高手如果有兴趣帮忙解决 请发邮件联系 Peter，非常感谢!
