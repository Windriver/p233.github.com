【分享】Sass for Sublime Text 2

刚刚修改了 nathos 的 sass-textmate-bundle （也就是 Package Control 中的 Sass 插件）作为自用，列出更改项目与大家分享。Github 下载地址。

2013 年 2 月 22 日更新记录：

  1. 添加了 @keyframes at-rule；
  2. 支持 @mixin @include 简写，scope selector: variable.shorthand.sass；
  3. 支持 Sass 自带函数，scope selector: support.function.buildin.sass；
  4. 移除了 none inherit auto 属性名，只做属性值使用；
  5. 给 transform 等属性添加了 -moz- 前缀；
  6. 给 resize 属性添加了 -webkit- 前缀；
  7. 移除了 snippets 文件夹。

不确定所有人都接受这些更改，希望各位 Sass 爱好者可以一起交流。
