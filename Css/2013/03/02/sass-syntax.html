【分享】Sass for Sublime Text 2

用过几个版本的 Sass 代码高亮插件，效果都不太理想，所以参考 nathos’s sass-textmate-bundle 自己也写了一个，简单列出新加入的特色功能，欢迎大家试用。

  1. 采用匹配语句结构的方式代替匹配关键词，因此不会出现某些词无法高亮的情况，但有些根据 Scope Selector 的 snippets 可能会失效；
  2. 支持属性选择器 (Attribute Selector) 高亮；
  3. 支持 Sass 专有语句，包括 Variable, Directive, Directive Shorthand (= +), Placeholder Selector, Build-in Function, Interpolation 等等；
  4. 所有 something(…) 将当做 function 结构处理，包括 url(), format(), 部分 CSS 属性值，Sass 自带函数，Compass/Bourbon 函数，自定义函数等等（注：函数的左括号在 Sublime Text 中无法更改颜色，Textmate 中可以，已经在官方论坛询问，可惜暂时还没有解决办法）；
  5. Sass 的 at-rule 当做 directive 结构处理，以后会进一步优化这部分的高亮效果；
  6. 优化了 @media 高亮效果； 7. 支持运算符高亮；
  7. 输入 id，class，placeholder 选择器时禁用 snippets；
  8. 分别支持 SCSS 与 Sass 两种语句结构，而不是将其合并在一起。

更多信息请查看 Github：https://github.com/P233/Syntax-highlighting-for-Sass，意见反馈欢迎联系我，大家一起交流！！
